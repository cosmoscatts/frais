## 技术栈
- 🤙🏻 Vue3 <setup>
- ⚡️ Vite2
- 🦾 TypeScript
- 🍍 Pinia
- 🔥 Vue-Router
- 🎨 Unocss
- 📥 unplugin-auto-import
- 🗒 unplugin-vue-components

## 功能点
* [x] 垂直、水平布局
* [x] 页面主题配置，包括『界面内容的显示』、『主题色更换』、『多页签配置』、『页面动画』等
* [x] 系统管理模块，包括『用户管理』、『角色管理』等
* [ ] 个人设置模块，包括『个人资料修改』、『修改密码』等
* [ ] 表格模块示例
* [ ] 图表模块示例
* [ ] 支持 `markdown`

## 如何启动？
```TypeScript
pnpm install
pnpm run dev
pnpm run build
```
