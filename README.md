# Petite

> Tips: 这是一个使用 vue3 搭配 Naive Ui 实现的一个简易后台管理系统模块，可以被用来当作是项目的脚手架。页面风格清新，可以自由配置。

*********

## 技术栈
- 🤙🏻 Vue3 <setup>
- ⚡️ Vite2
- 🦾 TypeScript
- 🍍 Pinia
- 🔥 Vue-Router
- 🎨 Unocss
- 📥 unplugin-auto-import
- 🗒 unplugin-vue-components

## 功能点
* [ ] 垂直、水平布局
* [ ] 页面主题配置
* [ ] 系统管理，用户管理 & 角色管理
* [ ] 支持 markdown

## 如何启动？
```TypeScript
pnpm install
pnpm run dev
pnpm run build
```
